코드의 구조는 어떻게 되어 있는가?
- 중첩된 코드인가? 
  - for loop 와 if 조합으로 조금 중첩된 코드
- 눈에 띄는 라인?
  - for loop 내의 조건별로 분기하는 if문. 조건이 좀 많아서 눈에 띔.

코드에서 들여쓰기는 어떻게 되어 있는가?
- 코드 간격? 
  - 적절하게 띄워져 있었던 것 같음.
- 큰 뭉치처럼 보이는 것?
  - 선언부, 각 메소드 별 한 덩어리로 보임. for loop가 한 덩어리로 보임
